Install packages

Please run the following code to install the packages needed. This needs to be run in your R console.

Code
### I may need mix Omics and impute not sure
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("mixOmics")
BiocManager::install("impute")
BiocManager::install("preprocessCore")
BiocManager::install("GO.db")

install.packages("remotes")
remotes::install_gitlab("YingxiaoYan/TriplotGUI")

Things may pop up in your console and ask you for some choices, we provide some recommedations here. We are conservative in the choices to avoid any unforseen conflict and complications in packages.

  • Sometimes it pops up in your console to ask you which package you would like to update. We recommened you to chose 3.None
  • Sometimes it pops up in your console to ask Do you want to install from soutce packages which need compliation . We recommended you to chose no
  • Sometimes it pops up in your console to ask Update all/some/none?[a/s/n] . We recommended you to chose n

Load libraries

Run the following code in your R console.

Code
library(TriplotGUI)

Download the example data for user interface

Go to my git repository. Click the download icon at the right side and download the Exampledata.rar. Unzip it and you will find the data that will be used as example in the tutorial.

Change the maximize size of data file allowed

The default of our package has allows data of maximize size of 5 MB. To change that, you could specify the maximize file size you prefer using the code below. Run your code in the console. However, we do not recommend to use data with more than 5 MB in our tool. The bigger the dataset, the slower it will be. It might be beneficial for you to do some variable selection before putting the data in out tool.

Code
filesize=100  ##MB
options(shiny.maxRequestSize=filesize*1024^2)

Activate the user interface

Run the following code in your R console.

Code
TriplotGUI_shiny_tryout()

Please run the following code to install the packages needed. This needs to be run in your R console.

```{r }
#| eval: false
#| echo: true
#| warning: false
#| error: true
#| classes: styled-output

if (!require("BiocManager", quietly = TRUE))

BiocManager::install("preprocessCore")

install.packages("remotes")
```
